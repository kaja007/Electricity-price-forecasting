# Electricity-price-forecasting
An end-to-end machine learning pipeline for forecasting electricity prices in the Day-Ahead Market (DAM) using time series feature engineering and gradient boosting models.

This project builds a production-style ML pipeline to predict electricity prices using historical market data. It covers:

- Data preprocessing from multiple CSV files  
- Time series feature engineering  
- Model training with hyperparameter tuning (Optuna)  
- Model evaluation and comparison  
- Ensemble modeling  


Raw CSVs → Preprocessing → Feature Engineering → Model Training → Evaluation → Ensemble
